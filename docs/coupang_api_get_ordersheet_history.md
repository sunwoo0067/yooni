# 배송상태 변경 히스토리 조회 (GET_ORDERSHEET_HISTORY)

특정 주문에 대한 배송상태 히스토리를 조회합니다.

## 요청

### Path

`GET /v2/providers/openapi/apis/api/v4/vendors/{vendorId}/ordersheets/{shipmentBoxId}/history`

### Example Endpoint

`https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/ordersheets/{shipmentBoxId}/history`

### Path Segment Parameter

| Name          | Required | Type   | Description                                         |
|---------------|----------|--------|-----------------------------------------------------|
| `vendorId`    | O        | String | 판매자 ID (쿠팡에서 업체에게 발급한 고유 코드) 예) A00012345 |
| `shipmentBoxId` | O        | Number | 묶음배송번호 (발주서 목록 조회 API를 통해 확인 가능) |

### Request Example

- 별도의 요청 본문(body)은 필요하지 않습니다.

## Error Spec

| HTTP 상태 코드(오류 유형) | 오류 메시지        | 해결 방법                                               |
|---------------------------|--------------------|---------------------------------------------------------|
| 400 (요청변수확인)        | Invalid vendor ID  | 판매자 ID(vendorId) 값을 올바로 입력했는지 확인합니다.    |
| 500 (요청변수확인)        | Internal Exception | 묶음배송번호(shipmentBoxId) 값을 올바로 입력했는지 확인합니다. |

# 출고중지완료 처리 (STOP_RETURN_REQUEST_SHIPMENT)

고객이 주문을 취소하여 반품요청목록 조회 시 상태가 출고중지요청(RELEASE_STOP_UNCHECKED) 이거나 반품접수미확인(RETURNS_UNCHECKED) 이면서 `releaseStatus`가 "N" 일 경우에 사용합니다. 아직 상품을 발송하지 않아, 고객 요청대로 상품을 출고하지 않았을 때 이 API를 호출하여 출고 중지 처리를 완료합니다.

**주의:** 출고 중지 요청 상품이 배송되면 왕복 반품 배송비는 판매자의 귀책입니다.

## 요청

### Path

`PATCH /v2/providers/openapi/apis/api/v4/vendors/{vendorId}/returnRequests/{receiptId}/stoppedShipment`

### Example Endpoint

`https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/returnRequests/363667/stoppedShipment`

### Path Segment Parameter

| Name        | Required | Type   | Description                                      |
|-------------|----------|--------|--------------------------------------------------|
| `vendorId`  | O        | String | 판매자 ID (쿠팡에서 업체에게 발급한 고유 코드) 예) A00012345 |
| `receiptId` | O        | Number | 접수 ID                                          |

### Body Parameter

| Name          | Required | Type   | Description                                      |
|---------------|----------|--------|--------------------------------------------------|
| `vendorId`    | O        | String | 판매자 ID (쿠팡에서 업체에게 발급한 고유 코드) 예) A00012345 |
| `receiptId`   | O        | Number | 접수 ID                                          |
| `cancelCount` | O        | Number | 출고 중지할 수량                                 |

### Request Example

```json
{
  "vendorId": "A00012345",
  "receiptId": 363667,
  "cancelCount": 2
}
```

## 응답

### Response Message

| Name          | Type   | Description                                |
|---------------|--------|--------------------------------------------|
| `code`        | Number | 서버 응답 코드                             |
| `message`     | String | 서버 응답 메세지                           |
| `data`        | Object | **`resultCode`**: `String` - SUCCESS/FAIL<br>**`resultMessage`**: `String` - 결과 메세지 |

### Response Example

```json
{
  "code": "200",
  "message": "OK",
  "data": {
    "resultCode": "SUCCESS",
    "resultMessage": "출고중지완료요청이 성공하였습니다"
  }
}
```
